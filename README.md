# Artemis II FAQ

Bienvenue sur le projet **Artemis II FAQ**, une application web interactive et immersive détaillant tout ce qu'il faut savoir sur la première mission lunaire habitée du programme Artemis II.

## Fonctionnalités 🚀
- **Modèle 3D interactif :** Explorez le vaisseau Orion en 3D (grâce à `@react-three/drei` et `@react-three/fiber`).
- **Expérience Immersive :** Animations fluides et arrière-plan spatial interactif (Starfield) dans l'en-tête.
- **Base de connaissances complète :** Les membres de l'équipage, la capsule, et tous les détails de la mission présentés dans une interface claire et futuriste.
- **Recherche Instantanée :** Trouvez facilement la réponse à n'importe quelle question sur la mission.

## Technologies Utilisées 🛠️
- **React, TypeScript & Vite**
- **Tailwind CSS** (design UI via Shadcn/Radix)
- **Three.js** (Rendu spatial et modèles 3D interactifs)

## Installation & Lancement 💻

1. **Installer les dépendances**
   ```bash
   npm install
   ```
2. **Lancer le serveur de développement**
   ```bash
   npm run dev
   ```

Explorez la mission et préparez-vous pour le décollage ! 🌌
